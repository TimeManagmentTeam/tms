1) Скачать проект https://github.com/TimeManagmentTeam/tms
2) Установить PostgreSQL https://www.postgresql.org/download/
При установке задать пароль q1w2e3R$ либо другой, но тогда не забыть поменять db-config.json
3) Установить Node.js и npm https://nodejs.org/en/ (npm (менеджер пакетов) - для установки пакетов react и ему сопутствующих, node.js - для работы этих пакетов).
Также нужно будет установить сами пакеты react и т.п. Для этого в папке Tms.WebUI нужно выполнить команду "npm install".
